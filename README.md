# Improved-overall-framework-diagram-of-adaptive-genetic-algorithm
flowchart TD
    A[算法开始] --> B[算法参数初始化<br/>种群规模N、进化代数上限Gmax、Pc/Pm上下限、多样性阈值、局部搜索迭代次数、罚函数系数上下限、编码规则、约束条件与目标函数]
    B --> C[初始种群生成<br/>采用混沌映射/正交设计/佳点集生成高多样性初始种群P(0)，设置进化代数g=0]
    C --> D[个体适应度与约束违反度计算<br/>1. 计算目标函数值<br/>2. 自适应罚函数处理约束条件<br/>3. 计算个体最终适应度值]
    D --> E[精英池更新与维护<br/>筛选当代Top-K精英个体，更新全局精英池，剔除重复解与劣解]
    E --> F[种群多维度状态评估<br/>1. 进化阶段划分：前期/中期/后期（基于g/Gmax）<br/>2. 适应度统计：f_max、f_avg、f_min<br/>3. 种群多样性评估：基于海明距离/信息熵计算多样性指数]
    F --> G{多样性是否低于设定阈值？}
    G -->|是| H[种群多样性增强策略<br/>自适应放大变异概率、劣解淘汰与随机移民补充、种群重组]
    G -->|否| I[自适应进化参数动态计算<br/>融合进化阶段、个体适应度、种群多样性，差异化计算每个个体的Pc、Pm、罚函数系数]
    H --> I
    I --> J[自适应选择操作<br/>基于进化阶段动态调整选择压力：前期低压力保多样性，后期高压力加速收敛]
    J --> K[自适应交叉操作<br/>按个体差异化Pc，匹配适配交叉算子（单点/两点/SBX交叉）生成子代]
    K --> L[自适应变异操作<br/>按个体差异化Pm，匹配适配变异算子（高斯/非均匀/位变异）优化子代]
    L --> M[精英个体局部搜索<br/>对精英池内个体执行2-opt/模拟退火/爬山法局部优化，强化局部寻优能力]
    M --> N[种群更新<br/>合并优化后精英个体与新生子代，采用精英保留策略生成下一代种群P(g+1)，g=g+1]
    N --> O{终止条件是否满足？<br/>g≥Gmax 或 最优解连续多代无改进}
    O -->|否| D
    O -->|是| P[输出全局最优解、最优适应度与优化结果]
    P --> Q[算法结束]
